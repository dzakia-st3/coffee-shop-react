# Coffee Shop App ☕

A simple and user-friendly web application for ordering food and beverages from a coffee shop.

Users can browse the menu, add items to their cart, and place orders easily. Authentication is required before placing an order.

## 🚀 Live Demo

https://coffee-shop-react-typescript.netlify.app/

## 🛠 Tech Stack

* React
* TypeScript
* Tailwind CSS

## ✨ Features

* User authentication (login & register)
* Browse menu items
* Add to cart & manage orders
* CRUD functionality
* Data stored using local storage

## 📌 Notes

This project uses browser local storage instead of a backend, so data is temporary and tied to the user's device.

## 📂 Getting Started

```bash
npm install
npm start
```
